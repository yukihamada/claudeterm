# ChatWeb — Product Hunt Launch Kit

## Tagline (60 chars)
Claude Code in your browser. Build, deploy, ship — just talk.

## Description (260 chars)
ChatWeb gives you a full AI development terminal in your browser. No install, no setup. Just type "build me a todo app" and watch Claude Code write files, run commands, and deploy — all in real-time. Free to start with $3 credit.

## Longer Description
ChatWeb is a browser-based development terminal powered by Claude Code. It's like having a senior developer sitting next to you — except it never sleeps, never judges, and can ship a full web app in under 5 minutes.

**What makes it different from Cursor/Copilot/Replit:**
- No install. Open chatweb.ai and start coding.
- Full terminal access — not just code completion. Claude reads files, writes code, runs tests, deploys.
- Workflow pipelines — step-by-step guides for Web, Mobile, Video, Research, Design projects.
- Live preview — see your app running as Claude builds it.
- Session sharing — share your work with a link, let others fork or collaborate.
- Template starters — React, Next.js, Rust API, Python ML, and more.

**Built with:**
Rust (axum) + SQLite + Claude Code CLI + Fly.io
